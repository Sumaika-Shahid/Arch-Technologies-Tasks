# Task 3: Number Guessing Game

This program allows the computer to randomly select a number between 1 and 100 and asks the user to guess the number.

# Features:
1. Generates a random number between 1 and 100
2. Allows the user to guess the number
3. Provides hints if the guess is too high or too low
4. Continues until the correct number is guessed

# Concepts Used:
1. Random number generation
2. Loops
3. Conditional statements
4. User input

# Language Used:
C++

# How to Run
Compile the program using any C++ compiler.
