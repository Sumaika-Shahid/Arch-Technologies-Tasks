# Arch-Technologies-Tasks
This repository contains solutions for two programming tasks implemented in C++.

##  Task 1: Dice Rolling Game

This program simulates rolling a pair of dice.

### Features:
- Generates two random numbers between 1 and 6
- Displays the result of both dice
- Shows the total
- Asks the user if they want to roll again

### Concepts Used:
- Random number generation
- Loops
- User input

---

##  Task 2: To-Do List Application

This program creates a simple To-Do List system.

### Features:
- Add tasks
- View all tasks
- Remove completed tasks
- Menu-driven interface

### Concepts Used:
- Vectors (Dynamic Array)
- Loops
- Conditional Statements
- User Input Handling

C++
