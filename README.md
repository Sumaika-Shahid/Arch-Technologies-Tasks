
# Arch Technologies Internship Tasks

This repository contains the programming tasks completed during my internship at Arch Technologies.
The purpose of these tasks is to strengthen programming fundamentals, problem-solving skills, and understanding of core programming concepts.


# Task 1: Dice Rolling Game

This program simulates the rolling of a pair of dice.
Each time the program runs, it randomly generates two numbers between 1 and 6, representing the result of each die. The program displays the results and asks the user if they would like to roll the dice again.

# Concepts Used

- Random number generation
- Loops
- Conditional statements


# Task 2: To-Do List Application

This program creates a simple To-Do List application where users can manage their daily tasks.
Users can add tasks, view the current list of tasks, and remove tasks once they are completed.

# Concepts Used

- Lists / Arrays
- Functions
- User input handling


# Task 3: Number Guessing Game

In this game, the computer randomly selects a number between 1 and 100.
The user must guess the number, and the program provides hints indicating whether the guess is too high or too low until the correct number is guessed.

# Concepts Used

- Random number generation
- Loops
- Conditional statements


# Task 4: ATM Simulation

This project simulates a basic ATM system.
It allows users to check their balance, deposit money, and withdraw funds. The project demonstrates basic Object-Oriented Programming (OOP) concepts by organizing ATM operations within classes.

# Concepts Used

- Classes and Objects
- Encapsulation
- User interaction handling


# Technologies Used

- C++
- Object-Oriented Programming (OOP)
